# Git branches

https://www.youtube.com/watch?v=e2IbNHi4uCI&t

## Commands

```bash
git branch <branch-name> # creates, but doesn't checkout to it
git checkout -b <branch-name> # creates and checks out to it

git checkout <other-branch> # switching branches
git branch -m <new-name> # rename current branch
git branch -m <old-name> <new-name> # rename other branches

git push -u origin <local-branch> # upload a branch


git branch --track <local-branch> origin/<remote-branch> # 'old' grabbing a remote branch
# modern
git fetch
git checkout <remote-branch> # creates local branch | points to remote | sets up tracking [so, git pull, push work without any arg]
git branch -vv # checking the link
```
